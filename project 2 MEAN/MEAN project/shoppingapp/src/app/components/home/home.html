Your one-stop shop for managing products with ease.

🚀 How to Run the Project

  1. Clone the repository: git clone https://github.com/dc152003/project2.git
  2. Open the project in VS Code: code .
  3. Open two terminals in VS Code.

🖥️ Terminal 1 — Backend

  • cd "MEAN project/shoppingdb"
  • node server.js
  • Ensure MongoDB is running and the shopping database exists.

🌐 Terminal 2 — Frontend

  • cd "MEAN project/shoppingapp"
  • npm install
  • ng serve
  • Open http://localhost:4200/

🛒 Features

  • View product list from MongoDB
  • Create new products
  • Update existing products
  • Delete products
